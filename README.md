# Simple 2D Unity Game

A top-down 2D action game built with Unity where players survive against waves of enemies in a dynamic environment.

## Features

### Player Mechanics

- **Shooting System**: Player can shoot bullets to defeat enemies
- **Health System**: Player health displayed in the HUD (heart icon with health counter)
- **Movement**: Top-down character movement controls
- **Camera Follow**: Camera automatically follows the player for smooth gameplay

### Enemy System

- **AI Behavior**: Enemies have intelligent AI that pursues and attacks the player
- **Random Spawning**: Enemies spawn randomly throughout the game
- **Health Bars**: Enemies display individual health bars above them

### Game Environment

- **Tilemap System**: Interactive tilemap with different terrain types
- **Damage Zones**: Special tiles that damage the player on contact
- **Visual Variety**: Includes grass, water, trees, and other environmental elements

### Visual & Audio

- **Animations**: Characters and objects have animated sprites
- **Sound Effects**: Audio feedback for game actions

### UI/HUD

- **Health Display**: Real-time player health monitoring
- **Score System**: Track player score in the top-right corner
- **Collectibles**: Gem/crystal items to collect for points

## Gameplay

Navigate through the environment, shoot enemies, collect gems, avoid damage zones, and survive as long as possible while racking up the highest score!
